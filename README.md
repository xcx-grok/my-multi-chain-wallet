🎓 MultiChain Wallet - 区块链毕业设计

一个极简但专业的**多链钱包**，支持 Ethereum Sepolia、BSC Testnet、Polygon Amoy。

## ✨ 功能
- 生成/导入 12 词助记词钱包
- 多链余额实时查询
- 测试币转账 + 交易历史记录
- 现代玻璃态 UI + 二维码
